假设检验原理1-t检验和F检验

2018-02-28
  1. 单样本均数的t检验
  2. 配对样本均数的t检验
  3. 两独立样本均数的t检验
  4. 两独立样本方差的齐性检验
  5. 问题遗留
  6. 参考声明

t检验用来检验两个小样本(通常<60)的均值差异是否明显。
前提条件：

  1. 理论上样本取自于符合正态分布的总体
  2. 要求组内样本的方差相等（称为方差齐性）

单样本均数的t检验

适用情况：只有一组数据X(均值为 μ 可计算)，总体均值 μ0 已知。求样本与总体有无差别

检验假设：
原假设： μ = μ0
备择假设：
μ ≠ μ0，双侧时
μ > μ0 或者 μ < μ00 ，单侧时

计算检验统计量：

  1. S 是样本标准差，v 是自由度，可计算出 t 值
  2. 查 t检验临界值表 比较 t 值计算 P 值：当 P > α 时接受原假设，否则拒绝原假设

配对样本均数的t检验

适用情况：两组数据形成对子，因此样本数相同。控制影响实验的无关变量。
优点：减少个体差异对实验的影响
缺点：数据成对处理

配对情况可以分为三类：

  1. 两个相似的个体的配成一对进行不同的处理得到两个数据
  2. 同一个个体同时进行两个处理得到两个数据：我的理解是对同一个个体的局部可以同时进行多个处理，比如人有两只手就可以在两只手上分别进行不同的处理得到两个结果
  3. 同一个个体在时间先后上进行记录得到两个数据

第一种是使用两个个体
第二种是使用一个个体同时间不同空间的数据
第三种是使用一个个体同空间不同时间的数据

检验假设：
零假设： μd = 0 (无差别)
备择假设：μd ≠ 0 (有差别)

计算检验统计量：

  1. 因为是配对数据（样本量为N），先计算 n 个差值，转化为对一组数据的处理
  2. 计算 n 个差值的均值 d
  3. 根据 n 个差值和均值 d 计算 Sd
  4. 根据均值 d 、样本量 n 和第二步的结果 Sd 计算 t 值和自由度 v
    t = |d|/(Sd/sqrt(n))
    v = n - 1
  5. 查 t检验临界值表 比较 t 值计算 P 值：当 P > α 时接受原假设，否则拒绝原假设

这是原PPT给出的 例子 和 解法

两独立样本均数的t检验

适用情况：将样本随机分为两个组，分别进行两种不同的处理（或者观察），因此每个样本只可能受到一种处理，而不会像“配对样本均数的t检验”那样——同一个样本可能同时（或者同空）地接受两种处理。两组数据无配对情况，因此样本数（例数）可以不同（分别为 n1 , n2 )

检验假设：
原假设：μ1 = μ2 (处理无效)
备择假设：μ1 ≠ μ2 (双侧，处理有效)

计算检验统计量：

  1. 计算自由度：
    v = n1 + n2 -2
  2. 分别计算出两组数据的均值 X1 和 X2 、方差 S12 和 S22
  3. 根据两组均值和方差计算 Sc2:
    [S12 ▪ (n1-1) + S22 ▪ (n2-1)] / v
  4. 计算 S X1 - X2：
    S X1 - X2 = sqrt(Sc2 ▪ (1/n1+1/n2))
  5. 计算 t :
    (| X1 - X2 |) / S X1 - X2
  6. 查 t检验临界值表 比较 t 值计算 P 值：当 P > α 时接受原假设，否则拒绝原假设

两独立样本方差的齐性检验

适用情况：在上述第三种情况（两独立样本均数的t检验）下，计算统计量 t 需要保证两组数据方差相同，因此需要对方差进行齐性检验。

检验假设：
原假设：σ1 = σ2 (方差齐性)
备择假设: σ1 ≠ σ2 (双侧，方差不齐)

计算检验统计量 F ：
F = max{S12, S22} / min{S12, S22}
v1 = n1 - 1
v2 = n2 - 1
查 α=0.05时的F临界值表 比较 F 值：当 计算值 > 查表值 时 P < 0.05 ,此时接受原假设（两组独立样本的方差的差别无统计学意义），否则拒绝原假设。
其他α值自行百度。
